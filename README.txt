README
Project is based on node.js, and of course you have to install it before.
This project used mongodb so you have to install it to.
All others decencies for node are described in package.json file.
To fill database use files in the directory dataset.
To run server use the command : node gomassSrv.js
To connect the default URL is : http://localhost:3333

Setup on localhost :
1-open command line and launch mongodb
C:\Users\David>mongod
2-open a node command line and launch node js server :
cd /d C:\Users\David\Documents\GitHub\gomass
node gomassSrv.js
3-open Firefox and launch game :
"C:\Program Files (x86)\Mozilla Firefox\firefox.exe" http://localhost:3333
